# Configuration And Observability

## Confirmed Configuration Areas

- language
- provider and model
- local endpoint/model
- API key storage mode
- generation mode
- ALT limits
- validation and QA
- signatures
- auto-upload behavior
- history
- debug
- on-complete action

## Confirmed Observability Areas

- local history
- debug log
- metrics by provider/model/source/mode
- support export
- config export/import
- config smoke test

## Main Caveat

- observability is browser-local only; there is no backend telemetry layer in this repo
