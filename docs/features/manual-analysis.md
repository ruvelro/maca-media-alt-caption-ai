# Manual Analysis

## Confirmed

- can be triggered from context menu on `wp-admin`
- can be triggered from keyboard shortcut `maca-run`
- opens overlay immediately in loading state
- supports regenerate variants:
  - default
  - technical
  - short
  - editorial
- supports copy of:
  - ALT
  - title
  - leyenda
  - all entries
- can apply values directly to visible WordPress fields
- can append active signature manually from overlay

## Inputs Used

- image URL
- filename or UI text context when available
- tab session context
- selected SEO profile
- section style profile
- provider/model settings
